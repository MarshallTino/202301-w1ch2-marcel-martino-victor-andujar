Isdi Coders

ISDI Coders es una coding school fundada en 2016 como una iniciativa de un equipo con 20 años de experiencia en el sector del empleo online. Nuestros cursos intensivos de programación informática se adecúan a las necesidades metodológicas, tecnológicas y laborales del mercado.

Bootcamp de Desarrollo Web Full-Stack

¿Quieres aprender las tecnologías y metodologías necesarias para realizar una aplicación web completa? En nuestros cursos intensivos full-stack aprenderás a programar front-end y back-end, con las herramientas, tecnologías y metodologías necesarias para comenzar tu carrera como desarrollador web. Se trata de cursos de 11 semanas de duración que se imparten presencialmente de lunes a viernes de 9h a 17h en nuestras sedes de Madrid, Barcelona y Málaga.

Bootcamp de Desarrollo Web Front-End

Para aquellos que necesiten combinar su actividad profesional con un programa que les introduzca y actualice en las metodologías y tecnologías de desarrollo web, hemos creado la versión part-time de nuestros cursos: durante 13 semanas (de jueves a sábado) prepararemos a nuestros estudiantes para posiciones de front-end developer.
